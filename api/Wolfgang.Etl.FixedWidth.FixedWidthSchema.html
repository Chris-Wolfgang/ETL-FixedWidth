A read-only view over the resolved fixed-width field layout for a record type (#22). Exposes the field/skip columns, positions, widths, types, and formatting that FixedWidthExtractor<TRecord> / FixedWidthLoader<TRecord> derive from [FixedWidthField] / [FixedWidthSkip] attributes — useful for generating documentation, building validation tooling, or debugging a layout.
